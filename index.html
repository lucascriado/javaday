Transformando o Ecossistema Tech Local

O Java Day é um evento anual focado em disseminar conhecimento técnico de alto nível, promover o networking e fortalecer a comunidade de desenvolvedores do Noroeste Paulista.

Organizado pela comunidade Java Noroeste e o IFSP, unimos academia e indústria para discutir as tendências de Java, Cloud Native, Microserviços e Engenharia de Software.
